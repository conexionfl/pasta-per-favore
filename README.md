# Pasta Per Favore — Website Mockup

Revamp mockup for [Pasta Per Favore](https://www.pastaperfavore.com), 3102 S 24th Street, Kansas City, KS 66106.

**Live preview:** https://conexionfl.github.io/pasta-per-favore/

- `index.html` — the full one-page site (menu, specials, gallery, story, catering, reviews, order flow, visit)
- `menu-print.html` / `menu-print.pdf` — print-ready menu (letter size), consistent with SpotOn online ordering
- `serve.py` — local preview server (`python3 serve.py`, then open http://localhost:8765)

All "Order Online" actions point to the SpotOn ordering page. Menu items and prices mirror SpotOn (source of truth).
